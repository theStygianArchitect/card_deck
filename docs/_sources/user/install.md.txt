# Installation of Deck of Cards

This part of the documentation covers the installation of Deck of Cards. The
first step to using any software package is getting it properly
installed.

## Install Deck of Cards via pip

To install Deck of Cards, simply run this simple command in your terminal of
choice:

```shell
$ python -m pip install git+https://github.com/theStygianArchitect/card_deck
```

## Install Deck of Cards via uv

To install Deck of Cards, simply run this simple command in your terminal of
choice:

```shell
$ uv add git+https://github.com/theStygianArchitect/card_deck.git
```


## Install Deck of Cards via poetry

To install Deck of Cards, simply run this simple command in your terminal of
choice:

```shell
$ poetry add git+https://github.com/theStygianArchitect/card_deck.git
```

## Install Deck of Cards via pipenv

To install Deck of Cards, simply run this simple command in your terminal of
choice:

```shell
$ pipenv install -e git+https://github.com/theStygianArchitect/card_deck.git
```

## Get the Source Code

Deck of Cards is actively developed on GitHub, where the code is [always
available](https://github.com/theStygianArchitect/card_deck).

You can clone the public repository.

```shell
$ git clone https://github.com/theStygianArchitect/card_deck.git
```
